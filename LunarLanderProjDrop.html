This project was developed from scratch using openframeworks in C++. The project models were also developed by me. The project incorporates OpenGL shaders and openframeworks lighting, the latter of which is similar to openGL lighting. The Lunar Lander project also included particle effects with physics and emitters along with an octree collision detection algorithm. This project is akin to a remake of the classic 'lunar lander' game which was bundled on atari game consoles in the early years of video game industry.

The tech stack for the application is as follows:

  • C++
  • OpenFrameworks
  • Visual Studio
  • OpenGL lighting
  • OpenGL shaders
  • Openframeworks Lighting
  • Particle effects
  • Octree Collision detection
  • Particle Emitters
  • Maya Model Development
  • Maya textures
  • Ray casting
  • Ray intersection
